Verification Quiz Discord Bot
Tags
Coding
My objective was to screen new members joining my Discord server, and I came up with this unique and brilliant idea—all thanks to my friend's suggestion. I developed a bot that administered three randomized science quizzes to new members, which should be completed in one minute for each question.
The program was built using discord.py v2, incorporating the button interaction feature, and with the help of ChatGPT, I was able to create something that I had never attempted before.
How it worked: The primary part of the code was a recursive class that iteratively retrieved question and answer data from a JSON list until the member successfully answered three questions.
This project was a huge success, and the bot works really well! Its success inspired me to add various other features that no other bot offers. Now, it has become the main bot on my Discord server, which I have named Lab Assistant.
